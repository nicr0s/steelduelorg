## About The Project

![Screen Shot](https://i.ytimg.com/vi/s5bd9KMSSW4/maxresdefault.jpg)

"Steel Duel" is an action-packed 2D fighting game project developed using the Python programming language and the Pygame library. This project aims to provide an engaging gameplay experience with dynamic characters, smooth animations, and intuitive controls.

Players can immerse themselves in thrilling combat scenarios controlling distinctive fighters, each with unique abilities and attack animations. The game features a head-to-head battle system, allowing players to engage in fast-paced fights, employing strategic moves and precise timing to defeat opponents.

The project showcases:

Customizable fighters with varied attack styles and special moves.
Fluid animations and responsive controls for an immersive gaming experience.
Engaging sound effects and visual elements that enhance the overall gameplay ambiance.
A user-friendly interface designed for an enjoyable gaming session.
"Steel Duel" is a testament to the developer's passion for delivering an entertaining and captivating gaming experience, combining intricate gameplay mechanics with visually appealing elements, promising hours of thrilling combat for players of all ages.

## Built With

The "Steel Duel" project is constructed primarily using Python along with the Pygame library, a powerful set of tools for creating games in Python. Pygame offers various functionalities for game development, including graphics rendering, sound manipulation, and event handling, making it an ideal choice for building interactive 2D games.

Key components and tools utilized in the development of "Steel Duel" include:

Python: A versatile and widely used programming language, providing a robust foundation for game logic and mechanics.
Pygame: A Python library offering essential modules and functions for game development, such as graphics rendering, audio playback, and event management.
Sprites and Images: Visual assets, including character sprites, backgrounds, and icons, designed to enhance the game's visual appeal and create an immersive experience.
Sound Effects: Engaging audio elements, including sword clashes, magical spells, and background music, contributing to the game's atmosphere and excitement.
Animation: Smooth and dynamic character animations, achieved through sprite sheets and frame-based rendering, bringing the fighters and their actions to life on-screen.
Combined, these elements form the foundation of "Steel Duel," enabling the creation of a captivating and enjoyable gaming experience with responsive controls, visually appealing graphics, and engaging audio elements.Here are a few examples.

## Usage

Launching the Game:
Run the main Python file (main.py) to start the game. Upon execution, the game window will appear, presenting the player with an immersive gaming environment.

Game Controls:

Player 1 Controls:

Move Left: A Key
Move Right: D Key
Jump: W Key
Attack 1: R Key
Attack 2: T Key
Player 2 Controls:

Move Left: LEFT ARROW Key
Move Right: RIGHT ARROW Key
Jump: UP ARROW Key
Attack 1: NUMPAD 1 Key
Attack 2: NUMPAD 2 Key
Gameplay:

Engage in dynamic battles between the two fighters, each possessing unique abilities and attacks.
Use the controls to maneuver your character, execute attacks, and defend against opponents' moves.
Win rounds by reducing your opponent's health to zero, scoring points for each victory.
Score Tracking:

The game keeps track of the score for each player, displayed on the screen during gameplay.
Scores are updated based on the outcome of each round, with points awarded for winning rounds.
Power-Ups and Abilities:

Activate power-ups like double damage or damage reduction by using specific key combinations.
Utilize these power-ups strategically to gain an advantage over your opponent during battles.
Round Reset:

After each round, a victory image will be displayed momentarily, and the game will reset for the next round after a brief cooldown period.
Exiting the Game:

To exit the game, close the game window or use the appropriate exit command based on your operating system.

Click on the "Fork" button in the top-right corner to create your copy of the repository.
